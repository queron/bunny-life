# Bunny Life 🥕 - Pixelated Bunny Farming Simulator

Welcome to **Bunny Life**, a cozy pixel-art 2D farming simulator where you play as the bright yellow bunny farmer with floppy ears!

![Yellow Bunny Farm](https://github.com/queron/bunny-life/raw/main/preview.jpg)

## 🌟 Features

- **Custom Yellow Bunny Character**: Pixel-art yellow coat, floppy drooping ears, cute black eyes, pink nose, and hopping animations modeled directly after the original drawing.
- **Top-Down Farming Mechanics**: Till grass with the Hoe, sow crop seeds, hydrate soil with the Watering Can, watch crops grow through distinct visual stages, and harvest fresh produce!
- **7 Crop Varieties**: Carrots 🥕, Golden Carrots 🌟, Radishes 🧅, Strawberries 🍓, Sunflowers 🌻, Watermelons 🍉, and Lucky Clovers 🍀.
- **Bunny Market Stall**: Trade harvested crops for Bunny Coins 🪙 to purchase seed packs, upgrade tools, and buy farm items.
- **Dynamic Day & Night System**: 24-minute game clock with morning dawn, bright afternoon sun, golden dusk, and starry night sky with glowing fireflies.
- **Stamina & Rest System**: Manage Bunny's energy bar and sleep in the cozy Bunny Burrow house to restore full stamina and advance to the next day.
- **Chiptune Audio Synthesizer**: 8-bit sound effects for hopping, tilling, watering, harvesting, coin sales, and background chiptune melody.

## 🎮 Game Controls

- **WASD / Arrow Keys**: Move Yellow Bunny up, down, left, right.
- **Space / E**: Use active tool / interact with tile directly in front.
- **1 - 6 Keys**: Select inventory quickbar slot.
- **Mouse Click**: Target and interact with any grid tile or open Market Stall.

## 🚀 Playing Locally

Simply open `index.html` in any web browser or host using a simple HTTP server:

```bash
npx serve -l 3000 .
```

Then visit `http://localhost:3000`.
